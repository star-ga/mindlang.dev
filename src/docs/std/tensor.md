# Tensor Library

<p style="font-size: 1.25rem; color: #475569; margin-bottom: 2rem;">
  High-performance operations for N-dimensional arrays.
</p>

### Operations
All operations support automatic differentiation (autograd).

* <code>matmul(a, b)</code>: Matrix multiplication.
* <code>conv2d(input, weight)</code>: 2D Convolution.
* <code>relu(x)</code>: Activation function.
